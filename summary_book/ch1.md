# Chapter1 영상처리 개요

## 1.1 영상처리란 무엇인가?
영상처리(image processing)는 입력된 영상을 어떤 목적을 위해 처리하는 기술이다.
수학적 연산을 이용해 화소들에 대해 변화를 줌

## 1.5 영상의 형성 과정
(x, y)가 좌표값이고 f()는 해당 위치의 밝기 값이다 f = i * r 인데 i는 물체에 비친 조명의 세기이고 r은 반사 계수로서 0과 1사이의 값을 가진다.
디지털 영상을 형성을 위해 표본화, 양자화란 두 단계가 필요하다.
표본화: 유한개의 화소의 수만큼 입력 값을 취하는 것
양자화: 표본화된 화소이 값을 디지털 화소로 표현하는 단계

## 1.6 디지털 영상의 표현과 영상처리
표본화 수의 따라 세로 M 가로 N 크기가 결정된다.
만약 k 비트로 양자화 하면 2^k 만큼 표현할 수 있다.
그러므로 한 화소에 필요한 저장 공간은 M x N x k 비트가 필요하다.
